메가박스 앱 접근 권한 안내

  • 위치(선택)

    위치기반 서비스를 이용하여 가까운 극장 안내

  • 주소록(선택)

    스토어 상품, 포인트 선물, 티켓나누기 사용 시

  • 전화(선택)

    고객센터 전화 걸기

  • 저장공간(선택)

    영화사진, 무비포스트, 포토카드, 프로필 사진 사용 시

  • 카메라(선택)

    바코드 스캔, 포토카드 제작

  • ‘선택 접근 권한’은 해당 기능을 사용할 때 허용이 필요하며, 허용 하지 않을 경우 해당 기능 외 다른 서비스는 이용이 가능합니다.
  • 접근권한 변경 방법 ‘디바이스 설정 > 애플리케이션 > 메가박스 > 권한’에서 각 권한 별 변경이 가능 합니다.
